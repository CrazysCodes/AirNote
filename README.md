# 口袋 V2 - 轻日程口袋

一个前端-only 的移动端网页小工具，用来替代微信文件传输助手里的临时备忘：

- 随手记录到今天
- 把事项放进本周的周几上午/下午/晚上
- 保存常用内容并一键复制
- 本地 `localStorage` 持久化
- 支持 JSON 导入/导出

## 使用方式

直接用浏览器打开 `index.html` 即可。

如果希望用本地服务预览，可以在项目目录执行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173
```

## 数据说明

- 数据默认保存在当前浏览器的 `localStorage`。
- 点击右上角 `⇩` 可导出 JSON 备份。
- 点击右上角 `⇧` 可导入 JSON 备份。
- V1 不会自动写本地文件，也不上传云端。

## V1 边界

- 不做账号、同步、AI。
- 不做提醒通知。
- 不做具体几点几分，只保留上午/下午/晚上。
- 循环只支持“每周循环”。
- 沉底只改变展示优先级，不自动删除。
